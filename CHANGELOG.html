Features

  • An icon can now be added to the Reforged line.
  • Added customization for Unique lines. Also filtering out redundant lines, such as when an item is both Unique-Equipped and Unique-Equipped: Category (1).
  • Required level ranges (such as those on heirlooms) can now be selectively hidden.

Changes

  • The Reset buttons in the Options menu will reset multiple options when appropriate.
  • Added more item level overrides for new Season of Discovery items.

Fixes

  • Ring runes are now recognized in Season of Discovery.
  • Fixed the option to Trim Space when rewording Armor, Bonus Armor, and Block.
  • Added more edge case checks for localizations.
  • Text cache has been re-enabled.
